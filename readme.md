# Two-Link Mechanism Simulation

This project simulates a planar two-link rotating mechanism.

The program:
- Simulates the motion of links AB and BC
- Computes position, velocity and acceleration of points B and C
- Calculates the axial force in link AB
- Generates plots for each scenario
- Creates an animation of the mechanism

## Files

two_link_assignment.py – main simulation code  
outputs/ – generated plots and animation  
explanation.pdf – short explanation of the approach

## How to run

Install requirements:

pip install numpy matplotlib pillow

Run the script:

python two_link_assignment.py
